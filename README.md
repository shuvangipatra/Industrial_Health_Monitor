# 🏭 Industrial Machine Failure Predictor

An AI-powered dashboard built to predict industrial equipment downtime before it happens.

## 📊 Performance
- **Accuracy:** 93%
- **Model:** Random Forest Classifier
- **Features:** 9 Real-time sensors (Temp, RPM, Pressure, etc.)

## 🛠️ Tech Stack
- Python (Scikit-Learn, Pandas)
- Streamlit (Frontend Dashboard)
- Pickle (Model Serialization)

## 🚀 How to Run
1. Clone the repo
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
